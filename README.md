# MamoPay
Frontend payment Admin dashboad

### Technology Used
 - Sass
 - React Next.js
 - React-hooks-forms


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
